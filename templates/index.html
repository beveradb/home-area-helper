Home Area Helper

Build property search filters/alerts based on target destination walk/bus/drive time and area deprivation.

Destination address for travel time calculations - e.g. workplace, city center, family home
Maximum journey time to consider for each method of transport, in minutes.
If you can't/won't travel by one of these methods, set it to 0 to exclude.
These are additive, so if you can reach somewhere by any of these it'll be included.
Minimum deprivation rank (1-10) to consider, based on public IMD data.
1 allows all areas, 10 allows only the top 10% least deprived areas.
